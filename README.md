# CodeAlpha Stock Portfolio Tracker

## Objective
Build a simple stock portfolio tracker using Python.

## Features
- User enters stock names and quantities.
- Uses a dictionary to store stock prices.
- Calculates total investment value.
- Saves portfolio summary to a text file.

## Technologies Used
- Python
- Google Colab

## Files
- Stock_Portfolio_Tracker.ipynb
- portfolio_summary.txt

## Output
The program displays the total investment value and saves the results in a text file.
